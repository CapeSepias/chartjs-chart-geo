# Chart.js Graphs
[![NPM Package][npm-image]][npm-url] [![Github Actions][github-actions-image]][github-actions-url]

Chart.js module for charting maps. Adding new chart types: `choropleth` and `bubbleMap`l

**Works only with Chart.js >= 2.8.0**
